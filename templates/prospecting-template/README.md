# Prospecting Template Skill

Founder-led outbound prospecting. Takes anchor companies, builds a ranked tier list of 25 peers, scores each for product fit, finds warm paths through LinkedIn + VouchFor, and writes pitch hooks personalized to what each person has actually said publicly.

See `skills/prospecting-template/SKILL.md` for the full workflow and `skills/prospecting-template/references/` for the ICP, pitch format, and connections configuration you need to fill in before first use.
