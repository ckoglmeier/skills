# AI Conversion Ops

AI-powered conversion rate optimization tools for analyzing and improving landing pages and lead magnets.

## Tools

### CRO Audit

Analyzes landing pages across 8 dimensions:

1. **Headline Clarity** — Is the value prop obvious in <5 seconds?
2. **CTA Effectiveness** — Specificity, placement, and copy quality
3. **Social Proof** — Testimonials, logos, case studies, review count
4. **Urgency** — Time-limited offers, scarcity signals
5. **Trust Signals** — Security badges, guarantees, credentials
6. **Form Friction** — Number of fields, progressive disclosure
7. **Mobile Experience** — Layout, tap targets, load behavior
8. **Page Speed** — Perceived and actual load performance

**Output:** Score (0–100), letter grade (A+ through F), priority fixes ranked by impact, and industry benchmarks across 8 supported sectors.

### Survey-to-Lead-Magnet Engine

Processes survey data to:
1. Cluster respondents by pain points and needs
2. Generate targeted lead magnet strategies per segment
3. Recommend format (checklist, guide, template, tool, etc.)
4. Score conversion potential per segment

Supports: Typeform, Google Forms, SurveyMonkey exports.

## Key Features

- No API keys or headless browser required
- Batch processing across multiple URLs
- Local-only analysis with opt-in telemetry
- Industry-specific benchmarking across 8 verticals

## Recommended Cadence

- **Weekly:** Audit top converting pages
- **Post-survey:** Run lead magnet engine after collecting survey data
- **Pre-launch:** Validate new landing pages before paid campaigns
- **Monthly:** Competitive benchmarking

## Privacy

Remote telemetry is opt-in only. No code, file paths, or repo content is ever collected. Usage logged locally to `~/.ai-marketing-skills/analytics/`.
